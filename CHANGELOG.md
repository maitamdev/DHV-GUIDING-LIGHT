# Changelog

All notable changes to this project will be documented in this file.

## [1.2.0] - 2024-02-15
### Added
- AI Mentor recommendation system with Gemini API
- Portfolio builder with CV export
- Meeting scheduling system
- Learning analytics dashboard
- New UI component library (Button, Card, Modal, etc.)
- Custom hooks collection (useDebounce, usePagination, etc.)
- Zustand stores for state management
- SEO optimization components
- Error boundary for crash recovery

### Changed
- Improved project structure with types, hooks, utils directories
- Enhanced navigation with search and notifications
- Refactored services layer with TypeScript types

## [1.1.0] - 2024-01-15
### Added
- Dark mode support
- Blog section
- Course roadmaps
- CV preview and download

## [1.0.0] - 2023-12-01
### Added
- Initial release
- Course catalog and enrollment
- Firebase authentication
- Mentor profiles and matching
- Student and instructor dashboards
